# IDCOFFICE

Multi Level Marketing, Crowdfunding Project written in PHP, CMS included for managing the Home Page


### Technologies Used
- PHP 7.x
- Grunt
- HTML, CSS, JS
- CoinPayments API
- PHPMailer
- MQSQL


### Setup
This project was built on top of the LAMP Stack, and so you will need the following installed: 

- Linux
- Apache
- MariaDB
- PHP
